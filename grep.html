GREP

El comando GREP se utiliza para buscar cadenas de texto dentro de un archivo o parrafo.

Parámetros:

-E, --extended-regexp

Patron es una expresión regular de una extension (ERE)

Otros parámetros:

-n

Imprime las coincidencias de texto junto con sus lineas en el documento

Ejemplos:

Mostrar archivo ordenado alfabéticamente (a-z).

sort myfile.txt

-!?

012345

abcdefg

cdefg

uvwxyz

xyz
